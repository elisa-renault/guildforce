## UI style rules

- Formality: use neutral/informal product tone ("du" form) consistently.
- Buttons: short imperative verbs (`Speichern`, `Abbrechen`, `Löschen`).
- Avoid mixed terms (`User` vs `Benutzer`): prefer `Benutzer` in admin contexts, `Mitglied` for guild contexts.
- Keep placeholders concise; avoid truncation on mobile.
- Preserve variable tokens exactly (`{{username}}`, `{{count}}`, etc.).
